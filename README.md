# Banner Editor

Готовый мини-сайт для создания форумных баннеров и промо-картинок.

## Что умеет

- менять размер баннера
- редактировать заголовок, подпись и плашку
- переключать цветовые темы
- скачивать готовый PNG
- быстро рандомизировать стиль

## Как опубликовать

1. Создай отдельный репозиторий на GitHub, например `banner-editor`
2. Загрузи в него файлы из этой папки:
   - `index.html`
   - `styles.css`
   - `script.js`
3. В `Settings -> Pages` выбери:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/ (root)`
4. После публикации ссылка будет вида:

```text
https://твой-логин.github.io/banner-editor/
```

## Как встроить в XenForo

Создай Custom BB Code и вставляй сайт через `iframe`, как делали для RP-checker.
